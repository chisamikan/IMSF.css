features

Light

CSSで表示させるので、JpegやPNG画像よりも軽快に表示できます。

Responsive

縦横比を変更しても模様が崩れず、拡大しても綺麗な表示を保てます。

SASS Include

SASSファイル(.scss)を同梱しており、変数でお好みの色を指定できます。

how to use

  1. CSSを読み込みます。
  2. divタグやspanタグ等で適当なボックスを作り、クラス指定します。
  3. うまい！（テーレッテレー♪
